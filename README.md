## 命名规则介绍

1. 不同学校对于thesis前校名缩写处理方式不同。如校名缩写最后一字母为t，t与thesis的t是否合并，均有例可援引。

## 排序规则

1. 已经被收录CTAN的优先，未收录的次之
2. 按照包名完整名称字母表顺序排序

3.排序将在每月第1个周日手动更新
